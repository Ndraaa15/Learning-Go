Parallelism dan Concurrency
* Parallelism adalah sebuah teknik untuk menjalankan beberapa tugas secara bersamaan, biasanya di lakukan pada multi-core processor.
* Parallel programming merupakan teknik yang digunakan untuk memecahkan suatu masalah dengan cara membaginya menjadi bagian kecil dan dijalankan secara bersamaan.
* Concurrency adalah kemampuan sebuah program untuk melakukan beberapa tugas secara bersamaan.
* Concurrency biasanya di lakukan pada single-core processor. (bergantian)

CPU Bound dan I/O Bound
* CPU Bound adalah sebuah program yang membutuhkan banyak prosesing power -> good for Parallelism.
* I/O Bound adalah sebuah program yang membutuhkan banyak input/output operation -> good for Concurrency.
